**Luna The voice Assistant**
Luna – The Voice Assistant is an AI-powered desktop voice assistant developed in Python to enhance English communication skills, especially for rural children. The assistant provides interactive voice-based learning through speech recognition, pronunciation practice, educational games, quizzes, motivational content, and health-related reminders.
By combining Natural Language Processing (NLP), Speech Recognition, and Text-to-Speech technologies, Luna offers an engaging and hands-free learning experience that encourages users to improve their spoken English with real-time interaction.

**Objectives**
•	Improve English speaking and listening skills.
•	Assist rural children in learning English through voice interaction.
•	Provide real-time speech recognition and spoken responses.
•	Make language learning fun through quizzes, games, jokes, and fun facts.
•	Encourage healthy lifestyle habits with reminders and activity tracking.

**Features**
•	Voice recognition for command input 
•	Text-to-speech voice responses 
•	English learning through interactive conversation 
•	Wikipedia search for quick information 
•	Plays YouTube songs via voice command

**Installation**
**Clone the repository**
git clone https://github.com/yourusername/Luna-Voice-Assistant.git
**Navigate to the project folder**
cd Luna-Voice-Assistant
**Install dependencies** 
pip install SpeechRecognition
pip install pyttsx3
pip install pywhatkit
pip install wikipedia
pip install pyjokes
pip install textblob
pip install nltk
pip install pyaudio
Download the required NLTK corpus

**Running the Project**
python luna.py
Speak the wake word:
Luna
Then give commands such as:
•	Play Shape of You
•	Tell me about India
•	Tell me a joke
•	Health tip
•	Motivation
•	Fun fact
•	Log exercise walking for 30 minutes
•	Step count 5000 steps
•	Log meal breakfast with 350 calories
•	Daily summary
•	Meditation guide
•	Stop
